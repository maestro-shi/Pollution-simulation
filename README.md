# Optimization Methods

Реализация и сравнение методов оптимизации для задач SLIM и тематического моделирования (PLSA).

## Часть 1 — SLIM (Sparse Linear Methods)

Рекомендательная система на основе аппроксимации матрицы пользователь-товар. Задача — найти матрицу весов W, такую что A ≈ AW, с L1 и L2 регуляризацией.

### Реализованные методы

| Метод | Стратегия |
|-------|-----------|
| Координатный спуск | Cyclic, Random, Greedy (Gauss–Southwell) |
| Проксимальный градиентный спуск | Шаг 1/λ_max |
| Mini-batch SGD | Случайная подвыборка пользователей |

## Часть 2 — Тематическое моделирование (PLSA)

Разложение матрицы документ-слово через скрытые темы. Максимизируется логарифм правдоподобия.

### Реализованные методы

| Метод | Описание |
|-------|----------|
| Alternating Gradient Descent | Попеременная оптимизация матриц Φ и Θ |
| EM-алгоритм | E-step: апостериорные вероятности тем, M-step: обновление Φ и Θ |

## Запуск

Открой `Optimization_methods.ipynb` в Jupyter Notebook или Google Colab и запусти ячейки по порядку.
